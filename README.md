
# 🃏 Klondike Solitaire - C++ Data Structures Project

A fully playable **console-based Klondike Solitaire** game implemented in C++ as a practical demonstration of core **Data Structure** concepts.

---

## 📌 Project Overview

This project implements the classic Klondike Solitaire game while showcasing the use of fundamental data structures learned in coursework:

- **Stacks** → Used for Deck and Foundation piles
- **Linked Lists** → Used for dynamic Tableau columns
- **Arrays / Vectors** → For card representation and shuffling
- **Queues** → For move handling and game operations

The game includes proper rule enforcement, move validation, win detection, and an undo feature.

---

## ✨ Features

- Complete Klondike Solitaire rules implementation
- Interactive console-based gameplay
- Real-time move validation
- Win condition detection
- Undo last move support
- Clean and colorful console output (Windows supported)
- Proper memory management using dynamic data structures

---

## 🛠️ Technologies & Data Structures Used

| Component          | Data Structure Used     |
|--------------------|-------------------------|
| Deck & Foundations | Stack                   |
| Tableau Columns    | Linked List             |
| Card Management    | Vector / Array          |
| Game Logic         | Custom classes + OOP    |
